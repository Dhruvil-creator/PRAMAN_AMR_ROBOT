Developer Environment & Hardware Dependencies
===============================================

This document lists optional and required dependencies for development and hardware testing.

Python packages (pip):
- RPi.GPIO (hardware PWM / GPIO)  -- on Raspberry Pi
- picamera2 (optional)           -- camera support on Raspberry Pi
- opencv-python-headless (optional) -- alternative camera backend
- numpy (optional, used by some signal processing)
- imageio (for Picamera2 frame writes)

Installation examples:
- System packages (Debian/Raspbian):
  sudo apt update && sudo apt install -y python3-pip python3-dev

- Python packages:
  pip3 install RPi.GPIO numpy imageio opencv-python-headless

Notes:
- The code is defensive: if hardware libraries are missing the server will run in simulator/mock mode.
- Run backend with root privileges for GPIO access: sudo ./start.sh
